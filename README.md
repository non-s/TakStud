# TakStud — Gestão Educacional Web

Sistema web de gestão educacional. Dashboard completo para professores, alunos e administradores.

**[→ Acessar o sistema](https://non-s.github.io/TakStud)**

## Funcionalidades

- **Dashboard** com estatísticas em tempo real
- **Alunos** — cadastro, edição e busca
- **Tarefas** — criação, conclusão e filtros (pendentes/concluídas)
- **Comunicados** — publicação e gerenciamento
- **Grade Horária** — visualização semanal
- **Multi-role** — Professor, Aluno e Admin
- Dados salvos no navegador (localStorage)

## Stack

HTML · CSS · JavaScript — puro, sem frameworks, sem build.
